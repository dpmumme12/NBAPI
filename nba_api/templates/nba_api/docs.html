Results will be returned in JSON format as follows

Search By Team

- Search for teams to get their current roster with this url "https://nbapi1.herokuapp.com/TeamSearch/{team}/showStats=false/apiKey=*****".
- Just put in your custom apiKey and your are good to go.
- Set showStats=true to return each players career stats as well.

Search By Player

- Search for players with this url "https://nbapi1.herokuapp.com/PlayerSearch/{String}/apiKey=*****".
- Searches by string input and returns all players who's name starts with the string.

Get Players Stats

- Get a players career stats with this url "https://nbapi1.herokuapp.com/statsSearch/{id}/apiKey=*****".
- Put in a players id number to return all of their stats.
